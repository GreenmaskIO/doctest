# CLI reference

_Added in the `0.2` line._

## `doctest run`

Run a suite.

| Flag | Description |
| --- | --- |
| `--config <path>` | Path to the YAML config (required). |
| `--watch` | Re-run on file changes. |
| `--verbose` | Emit per-file diagnostics. |

## `doctest version`

Print the installed version and exit.
